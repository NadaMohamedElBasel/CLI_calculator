Usage Guide
===========

To use the calculator:

.. code-block:: python


   import py_calculator
   result = py_calculator.add(10, 5)
   print(result)  # 15

Supported functions:

- add(a, b)
- sub(a, b)
- mul(a, b)
- divide(a, b)

.. code-block:: bash

   CLI_calculator add 2 3
   CLI_calculator sub 5 3
   CLI_calculator mul 2 3
   CLI_calculator divide 10.0 5.0
